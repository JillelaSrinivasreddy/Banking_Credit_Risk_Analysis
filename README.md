# Banking Credit Risk Analysis Dashboard

## Project Overview
This project analyzes customer credit risk using the German Credit dataset. The dashboard helps identify good and bad risk customers to support better lending decisions.

## Tools Used
- Excel
- MySQL
- Power BI
- VS Code

## Key Insights
- 70% customers belong to Good Risk category.
- Car loans had the highest number of applicants.
- Customers with owned housing showed better risk profiles.

## Visualizations
- Risk Distribution (Pie Chart)
- Risk by Loan Purpose (Stacked Bar Chart)
- Risk by Housing Type (Stacked Column Chart)
